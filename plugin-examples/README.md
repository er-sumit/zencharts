# Zen Charts Plugin Examples

This folder contains a collection of example plugins designed to extend the functionality of Zen Charts and inspire the development of your own plugins.

## Running Locally

To run these examples locally, follow these steps:

1. Build the library in the root directory:
   ```shell
   npm install
   npm run build:prod
   ```

2. Switch to the Plugin Examples folder, install, and start the development server:
   ```shell
   cd plugin-examples
   npm install
   npm run dev
   ```

3. Visit `localhost:5173` in the browser.
