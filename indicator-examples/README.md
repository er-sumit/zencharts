# Zen Charts Indicator Examples

This directory contains implementations of indicator calculations and examples of how they can be shown on a chart.

## Running Locally

To run these examples locally follow these steps:

1. Build the library in the root directory:
   ```shell
   npm install
   npm run build:prod
   ```

2. Switch to the Indicator Examples folder, install dependencies, and start the development server:
   ```shell
   cd indicator-examples
   npm install
   npm run dev
   ```

3. Visit `localhost:5173` in the browser.
